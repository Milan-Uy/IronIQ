Core User Flows

The key interactions that define the app experience

1. Step-by-Step Plan Creation (Coach Tab)

User opens Coach
→
"Create a workout plan"
→
AI asks: split type?
→
AI asks: days/week?
→
AI asks: specialization?
→
AI generates full program
→
Saved to Program tab

User can ask follow-ups: "add more chest volume", "swap barbell for dumbbell", etc.

2. Exercise Substitution (Track Tab)

Logging workout
→
Tap "Swap" on exercise
→
AI shows alternatives
→
Pick one (with reasoning)
→
Logged as substitution

AI suggests alternatives based on same muscle group, available equipment, and biomechanical similarity.

3. Workout Comparison (Track Tab)

View workout history
→
Select two sessions
→
Side-by-side comparison
→
Volume/intensity delta

Shows: total volume change, weight progression per exercise, sets completed vs prescribed.

4. AI Insights (Insights Tab)

Open Insights tab
→
AI analyzes recent data
→
Summary cards
→
Trends + recommendations

Weekly/monthly summaries, muscle group volume balance, progressive overload tracking, plateau detection.

5. Browse & Adopt Templates (Program Tab)

Open Program tab
→
Browse templates
→
Preview PPL / UL / FB
→
Adopt & customize
→
Set as active program

Templates are copied to user's programs. Can then customize via Coach or direct editing.
