Provider Abstraction Layer

A unified interface wrapping both Claude and OpenAI:

interface AIProvider { chat(messages, systemPrompt, tools) → response structuredOutput(messages, schema) → JSON } // Implementations: ClaudeProvider — uses @anthropic-ai/sdk OpenAIProvider — uses openai SDK // Switch via env var: AI_PROVIDER=claude|openai

System Prompt — AI Coach Persona

The AI is prompted as a knowledgeable fitness coach with expertise in:

  • Exercise programming & periodization
  • Biomechanics & movement patterns
  • Exercise alternatives based on equipment/injury/preference
  • Progressive overload strategies

Context injected per request: user profile, current program, recent workout history

AI Tool Functions (Structured Actions)

The AI can call these tools to modify data:

create_program(name, split, days, exercises) modify_exercise(day_id, exercise, changes) suggest_alternative(exercise, reason, constraints) compare_workouts(session_id_1, session_id_2) generate_insight(type, date_range) create_plan_step_by_step(preferences)
